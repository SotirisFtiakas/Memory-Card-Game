# Memory-Card-Game
A Memory Card Game (Super Mario Theme) versus Players/Bots
_________
### Bot Levels

* **Easy** - Goldfish (no memory)
* **Medium** - Kangaroo (remembers half of the cards opened)
* **Hard** - Elephant (remembers all of the cards opened)
